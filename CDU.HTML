¿Qué es CDU?

El Diseño centrado en el usuario es una técnica utilizada para el enfoque en el usuario y las necesidades que requieren durante el proceso.

Esto significa que se debe tener en cuenta la experiencia deol usuario a la hora de diseñar.

Etapas de CDU

  • Entender el contexto del usuario
  • Especificar los requerimientos del usuario
  • Diseño
  • Evaluar

1.Entender el contexto del usuario

El objetivo es comprender que es lo que el usuario desea para así brindarles la página de la manera acertada con todo lo solicitado.

2.Especificar los requerimientos del usuario

Identificar y especificar los requisitos del usuario, como el tipo de diapositivas para acceder, videos de intrucciones en lugar de guías.

3.Diseño de soluciones

Los diseñadores crean soluciones para las necesidades del usuario y adaptarse a lo soliciten

Es importante informar cualquier cambio tanto el creador como para el usuario

4.Evaluar resultados

Evalúan los resultados y requisitos del usuario para verificar que tan buen desempeño tuvo el diseñador.

Lo que indica que revisan que tan parecido fue el resultado a lo solicitado.

beneficios de DCU

  1. Aumenta la satisfacción del usuario.
  2. Reduce costos de soporte,mantenimiento y desarrolo.
  3. Aumeta productividad y eficiencia.
  4. Incrementa el uso del sistema.
